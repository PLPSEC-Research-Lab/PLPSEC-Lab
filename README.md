# PLPSEC-Lab
# Water Treatment Plant MQTT Telemetry Lab
 
## Objective
Research and simulation of MQTT-based telemetry in water treatment systems and detection of cyber attacks in industrial environments.
 
## Architecture
- Sensors (water level, pressure, flow)
- MQTT Broker
- Monitoring Layer (Wazuh + Grafana)
- Attack Simulation Layer
 
## Protocol
MQTT (Message Queuing Telemetry Transport)
 
## Research Focus
- Data integrity attacks
- MQTT spoofing
- Unauthorized topic subscription
- Replay attacks
- Anomaly detection in telemetry streams
 
## Environment
- Linux (Kali / Ubuntu)
- Mosquitto MQTT Broker
- Wazuh SIEM
- Grafana
 
## Purpose
Academic research, cybersecurity defense, industrial monitoring.
